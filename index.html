Defining a roadmap towards a Web Development career.

As a journey to become a Web Developer, there are 3 paths or roadmaps to follow, Front-end, Back-end & DevOps. The three paths provide you the tools to start building real projects.

Let's Check them>>

Reasons to choose a Web Developer career. Connecting the dots...

There could be a lot of reasons why you would want to code & get into web development, I will tell you my reasons, I'm sure you have yours!

  • I wish to be part of the technological change which it is impacting people's lives contributing to build web & mobile apps.

  • Learning never stops. Web tools & technologies keep & will keep on changing for good, there is no way to get bored.

  • Web Development Industry has a high demand for skilled people in tools & web technologies.

  • No matter where I come from or go in the world, I can apply those skills whether I decide to work as a Freelancer or to apply for a job position inside a small, medium or big company.

HTML, CSS, JS, PHP, where & how to start? A roadmap is not clear at the beginning.

If you are reading this, I'm sure you are or were confused at the very begining, listening to stuff like markup language, styles, programming languages, etc. After a research & couple of days looking into the diferent ways to learn to code I found out about FreeCodeCamp where you can learn to code and help Nonprofits. There are of course other alternatives free & paid ways to learn to code, however, I think Free code camp is a good way to start for several reason:



  • It's Free & self paced. However,if you can afford to Donate & support this great project. I will encourage you to do it, It is the only way to keep teaching people to code for free specially for those that could could not afford any other media of learning.

  • Its curriculum makes you put into practice all learnt stuff in its lessons, building projects that at the end of the day proof your skills first to yourself, keeping you self motivated & then to any possible employer, letting him to watch that you indeed have experience on building real life projects.

  • It has a Support Community via Chat, a facebook community group in your home town that can help you to solve inquiries and providing feedback while performing the challenges of building the projects.

  • You can get 1 to 4 certifiactions that has recognization & can get you into Web Development bussiness if you are serious. Front End Development, Data Visualization, Back End Development & Full Stack Development Certification.

Roadmap to become a Web Developer Choosing your path.

As you can appreciate at this Roadmap. there are 3 paths to become a web developer. Front-end, Back-end & DevOps. Basically 3 paths that provide you the necessary tools to start building projects.


Front-end

  • HTML
  • CSS
  • JavaScript
  • Frameworks: Bootstrap, Foundation
  • Preprocessors: Sass, Less, Stylus
  • Task Runners: npm scripts, Gulp, Grunt
  • Package Manager: Yarn, npm

Back-end

  • Languages: Ruby, Python, Node Js, PHP 7
  • Ruby: RubyGems, RSpec, Ruby on Rails
  • Python: Pip, unittest/pyUnit, Django, oiohttp
  • Node Js: Jest, Mocho, Jasmine, Express, npm, Yarn
  • PHP 7: Composer, PSRs, Loravel, PHPUnit, phpspec

DevOps

  • Operating System: Unix, Linux
  • Automation: Puppet, Ansible, AWS Cloud Formation, Terraform
  • Cloud: Cloud Foundry, Rackspace, AWS
  • CI / CD: Jenkins, Travis, TeamCity
  • Web Servers: Apache, Nginx


I started following the Front-end path and even I have already built my first Responsive Web site. I expect this to be this just the begining to grow my personal portfolio. As probably you already know the path is not always easy, learning & putting into practice all the knowledge can be frustrating soemtimes, effort & self-motivation is needed from the very begining, but as long as you keep on the roadmap, the path starts becoming easier until you realize that you are able to build complete projects, yes fellows, We are able to achieve our dreams, just do not give up!

I have my goals clear, I want to get first the Front End Development certification and then go on with other 3 certifications: Data Visualization, Back End Development & Full Stack Development Certification. But at the end of the day, I also want to have Fun, because if you enjoy coding, your path towards a Web Developer will be easier & happier.
